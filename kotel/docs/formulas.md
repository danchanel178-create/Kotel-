## Ключевые соотношения (как в коде)

### Располагаемое тепло

- Твёрдое/жидкое: `Q_p_p = Q_i_p + i_tl + Q_v_vn + ΔQ_razm`
- Газообразное: `Q_p_p = Q_i_p + i_tl + ΔQ_razm`

### Потери

- `q2` — от энтальпии уходящих газов с учётом α, β и поправки на q₄
- `q4` — через среднее содержание горючего в шлаке/отложениях/уносе и зольность
- `q5_nom` — интерполяция кривой рис. 5.1 по D_ном (или ручной override)
- `η_k = 100 − (q2 + q3 + q4 + q5 + q6 + q5охл)`

### Расход топлива

```
B = Q_k / (Q_i_p · η_k/100 + Q_v_vn + Q_φ)
B_p = B_corrected · (1 − q4/100)
```

где `Q_k` по умолчанию равен `Q_poln` (5-16), либо задаётся вручную.

## Рекомендации после расчёта

Метод `_build_recommendations` анализирует q₂, q₃, q₄, нагрузку D/D_ном, влажность, α_ух и ηₖ и формирует текстовые советы для вкладки результатов.

## Список использованных формул

В `HeatBalanceResult.formulas_used` накапливаются номера формул, реально применённых при текущем наборе опций (например, 5-04 только при размораживании).
