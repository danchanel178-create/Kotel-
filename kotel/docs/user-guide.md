# Руководство пользователя

## Назначение

Приложение **Тепловой баланс котла** считает тепловой баланс парового котла: располагаемое тепло топлива, статьи потерь, КПД, полезную тепловую мощность и расход топлива.

Расчёты соответствуют формулам главы 5 (5-01 — 5-24).

## Запуск

1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите `python main.py` или `run.bat`

Рекомендуется Windows 10/11 (есть оптимизации DPI и перетаскивания окна).

## Разделы интерфейса

Навигация слева — шесть разделов.

### 1. Топливо

- Выбор **пресета** из справочника (газ, мазут, угли и др.)
- Тип топлива: твёрдое/жидкое или газообразное (влияет на формулу Qₚᵖ)
- Низшая теплота сгорания Qᵢᵖ, зольность Aᵖ, влажность, теплоёмкость, (CO₂)карб
- Опция учёта физического тепла топлива iₜл
- Параметры шлака и уноса для q₄ и q₆ (доли золы, содержание горючего, температура шлака)

### 2. Уходящие газы

- Энтальпии I_ух, I_х.в⁰, I_г.в
- Коэффициент избытка воздуха α_ух
- Доля рециркуляции воздуха β
- Потери от химической неполноты q₃ (%), задаются вручную

### 3. Котёл

- Номинальная и фактическая паропроизводительность D_ном, D (т/ч)
- Ручное переопределение q₅ном (иначе — интерполяция по рис. 5.1)
- Площадь неохлаждаемых поверхностей H_неохл
- Тепло с подогретым воздухом Q_в.вн (для твёрдого/жидкого топлива)

### 4. Полезное тепло

Параметры для Qполн (формула 5-16):

- Расходы и энтальпии перегретого пара, питательной воды, продувки
- Пар через пароперегреватель, отборы тепла

Энтальпии i_пе, i_пв, i', i_φ можно получить из плагина **Свойства пара** и подставить в форму одной кнопкой.

### 5. Расход топлива

Опции расчёта B (5-17 — 5-24):

- Переопределение Q_к
- Тепло внешнего воздуха (5-20)
- Паровой дутьё / впрыск (5-21)

### 6. Результаты

После нажатия кнопки расчёта отображаются:

- Метрики: ηₖ, Qполн, B, Bₚ, Σq
- Диаграмма потерь (столбцы q₂…q₆)
- Таблица полей результата
- Список использованных формул
- Практические рекомендации (предупреждения по режиму)

## Меню и действия

| Действие | Описание |
|----------|----------|
| Сохранить | Запись входных данных (и результата, если был) в JSON |
| Загрузить | Восстановление полей из ранее сохранённого JSON |
| Утилиты | Открытие плагинов (например, «Свойства пара») |
| Настройки | Режим подписей полей (полные / компактные), сведения о программе |

## Настройки

Файл `settings.json` в корне проекта:

| Параметр | Значения | Описание |
|----------|----------|----------|
| `label_mode` | `full` / `compact` | Полные подписи (символ + описание + единица) или компактные (символ + единица) |

## Типовой сценарий

1. Выберите топливо из справочника или введите характеристики вручную.
2. Заполните уходящие газы и параметры котла.
3. При необходимости откройте **Свойства пара**, рассчитайте энтальпии и подставьте их в баланс.
4. Укажите полезные тепловые нагрузки.
5. Нажмите расчёт и перейдите в **Результаты**.
6. При необходимости сохраните сессию в JSON.

## Требования к данным

- Qᵢᵖ и располагаемое тепло должны быть положительными.
- D > 0 для корректного пересчёта q₅ по нагрузке.
- Для расхода топлива B нужны Q_к > 0 и знаменатель формулы 5-19 ≠ 0.

При некорректных данных приложение показывает рекомендации на вкладке результатов, а не падает с ошибкой.
